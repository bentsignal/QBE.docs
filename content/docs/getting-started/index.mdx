## Installation

1. Download and install the [prerequisites](/getting-started/prereqs)
2. Set up your [local environment](/getting-started/local)
3. Deploy the web app to [Vercel](/getting-started/deploy-app)
4. Deploy the backend to [Convex](/getting-started/deploy-backend)
5. Set up [environment variables](/getting-started/env)
6. Set up [authentication](/getting-started/auth)
7. Set up [billing](/getting-started/billing)
8. Set up the [last few things](/getting-started/fin)
