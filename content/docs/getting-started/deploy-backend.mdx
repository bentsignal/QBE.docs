## Setup

1. Login or sign up for a [Convex](https://www.convex.dev/) account
2. Create a new project
3. Install the Convex CLI
   ```bash
   pnpm add -g convex
   ```
4. Login to your Convex account
   ```bash
   pnpx convex login
   ```
5. Initialize your development environment
   ```bash
   pnpx convex dev
   ```
   _Note_: This should fail due to missing environment variables. Once it fails, hit `Ctrl + C` to stop the server.
6. Commit and push your changes
